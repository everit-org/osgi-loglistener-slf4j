OSGi LogListener SLF4J

Introduction

An OSGi LogListener implementation that forwards the LogEntry to an SLF4J logger. This bundle contains a BundleActivator that opens a ServiceTracker for LogReaderServices to add the Slf4jLogListener the them.

A blog post is available on Everit - Opensource about this bundle.
